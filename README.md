# admissions-management-system
这是一个简单的公司公共资产管理系统，是我大学期间所做的毕业设计。本管理系统是建立在Dreamweaver上用php语言进行编写的，结合MySQL和集成软件WampServer实现基本的管理功能。本系统主要包含了“登录模块”、“管理员帐号管理模块”、“注册用户管理模块”、“公共资产管理模块”、“用户申请管理模块”、“系统管理模块”等功能模块。
